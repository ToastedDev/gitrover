# gitrover

The better [GitHub CLI](https://github.com/cli/cli) we all needed.

## Installation

```bash
# npm
npm install -g gitrover

# yarn
yarn add -g gitrover

# pnpm
pnpm add -g gitrover

# bun (not tested)
bun add -g gitrover
```
