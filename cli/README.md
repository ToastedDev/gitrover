# gitrover

> [!CAUTION]
> This is very work in progress. Don't expect _everything_ to work 100%.

The better [GitHub CLI](https://github.com/cli/cli) we all needed.

## Installation

```bash
# npm
npm install -g gitrover

# yarn
yarn add -g gitrover

# pnpm
pnpm add -g gitrover

# bun
bun add -g gitrover
```
