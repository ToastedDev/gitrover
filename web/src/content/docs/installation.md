GitRover publishes itself as a package on the [npm registry](https://npmjs.com). To install it, run any of the following commands in your terminal:

## npm

```bash
npm install -g gitrover
```

## yarn

```bash
yarn add -g gitrover
```

## pnpm

```bash
pnpm add -g gitrover
```

## bun

```bash
bun add -g gitrover
```
